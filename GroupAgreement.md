Our goals a group are to collaborate effectively to learn and complete the class material together, ensuring that each group member follows along at the same rate.

We all agree to the folowing expectations for group work: Allow each member to participate, Make sure each member puts in effort on shared work, Make sure to not hesitate to ask questions so that all people are on the same page

If we use generative AI in class, we agree to make a note that we used it, not copy and paste directly from the AI, using it to learn constructively and not as a crutch

If we are going to miss or be late to class, we agree to commute via text in advance.
